Parallel Coordinates by using Processing

Parallel coordinates (Wikipedia)

Paralle Coordinates is one way to visualize multi-dimensional data.

Acknowledgement:
The dataset is from UCI Machine Learning Repository: Iris Data Set.

  • Mouseover to a line to show its feature values.
  • Click on a feature name to select.
  • Move a knob to change the range.
